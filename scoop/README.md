# Scoop bucket

This directory is the Scoop bucket for `itonics`. The manifest at `itonics.json`
is **regenerated by GoReleaser on every release** (see the `scoops:` block in
[`.goreleaser.yaml`](../.goreleaser.yaml)).

## Install

```powershell
scoop bucket add itonics https://github.com/itonics/itonics-cli
scoop install itonics
```

`scoop update itonics` pulls the next release automatically.

Until the first `v*` tag is cut, this manifest does not exist yet — use the
[direct Windows download](https://github.com/itonics/itonics-cli/releases/latest)
in the meantime.
